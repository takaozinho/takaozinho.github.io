3.3 アメリカ企業のマネジメント


3.3.1 計画

　予算：売上－費用＝利益（見込み・目標）　年間予算、月次予算 　中・長期計画　中期（3-5年）長期（10年） 　生産販売計画⇒投資計画・事業展開（戦略） 　1980年代　コンサルティング業発展（財務会計・税務、生産、情報システム） 　新しい経営手法を積極的に導入 　他国の傾向　助言サービスが必要ないのでコンサルティング業の発展速度は遅い

3.3.3 指揮・調整

  • 平等主義的

  • トップダウン式

この組み合わせは例外的

  • 契約社会

  • ジョブ型

    • 職務は雇用契約時の職務指示書に記載されている。

　- 直接的表現　契約書が詳細に作成される（低文脈社会）　雇用契約・業績評価が詳細になる

  • 業績評価

    • ネガティブ・フィードバックは直接的表現をとらない。（Meyer (2014, 日本語版 p.)）

    • まず肯定・評価してから、改善点を付け加える

3.3.4 統制

　株式会社　出資者が株式を保有　株主が株主総会で取締役を選任　取締役は取締役会で経営をチェック　代表取締役は社長のことが多い 　外部統制　敵対的買収による経営解任のリスクが高まる 　内部統制　取締役（会）の役割：所有者（株主）に代わって、マネジメント・統制する→雇用、助言、評価、報酬、人事（管理職の配転） 　図表 　買収されるほど株価が下落しないように経営努力する←統制
